# ➗ Fraction Calculator

**Free online fraction calculator with step-by-step working** — add, subtract, multiply and divide fractions instantly. No ads, no signup.

🔗 **Live site:** [ronaldolum.github.io/fraction-calculator](https://ronaldolum.github.io/fraction-calculator)

---

## What can it do?

- ✅ **Add fractions** — e.g. 1/2 + 1/3 = 5/6
- ✅ **Subtract fractions** — e.g. 3/4 − 1/4 = 1/2
- ✅ **Multiply fractions** — e.g. 2/3 × 3/4 = 1/2
- ✅ **Divide fractions** — e.g. 3/4 ÷ 3/8 = 2
- ✅ **Simplify / reduce fractions** — e.g. 6/9 → 2/3
- ✅ **Step-by-step working** — shows every step including LCD, conversion, and simplification
- ✅ **Mixed number display** — shows results as mixed numbers where applicable
- ✅ **Auto-simplifies** all results to lowest terms
- ✅ Light and dark mode
- ✅ Works on mobile, tablet, and desktop
- ✅ No dependencies — pure HTML/CSS/JS

## How to use locally

Download `index.html` and open it in any browser — no install or server needed.

```bash
git clone https://github.com/ronaldolum/fraction-calculator.git
cd fraction-calculator
open index.html
```

## Deploy your own

1. Fork this repo
2. Go to **Settings → Pages**
3. Set source to `main` branch, `/ (root)` folder
4. Your site is live at `yourusername.github.io/fraction-calculator`
